 >> PENERAPAN FORM LANJUTAN <<


 >> <textarea></textarea> = Mirip textfield, ukuran lebih besar dan tulisan bisa 
			    disimpan lebih banyak(artikel, testimoni, komentar)

 >> Dropdown, penerapannya menggunakan tag select yg di dlmnya terdapat tag 
    option

    => Contoh dropdown dan textfield : 

		<textfield></textfield>
		<br>
		<select>
			<option>Bandung</option>
			<option>Yogyakarta</option>
			<option>Jakarta</option>
		</select>

 >> Cara mengakali agar form terlihat lebih rapih hanya menggunakan HTML, yaitu 
    dgn menggunakan tag table
